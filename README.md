# Chef.in

Добро пожаловать в Chef.in — веб-приложение, которое помогает пользователям находить, делиться и обсуждать кулинарные рецепты. Этот проект был разработан в рамках  nFactorial Incubator.


## Начало работы

Эти инструкции помогут вам запустить копию проекта на локальной машине для разработки и тестирования.

### Предварительные требования

Что нужно установить для работы с программой:

- Node.js
- npm (Node Package Manager)

### Установка

Пошаговое руководство по запуску среды разработки:

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yelnurdo/cookingAppNfactorial.git
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd cookingAppNfactorial
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Запустите сервер разработки:
   ```bash
   npm start
   ```

Приложение теперь должно быть доступно на [http://localhost:3000](http://localhost:3000).

## Уникальные подходы
- Интеграция с внешним API для получения данных о рецептах. Ранее рассматривался API Edamam, однако было решено использовать API от TheMealDB, так как он предоставляет более широкий выбор рецептов и удобную структуру данных.

## Использование

После установки вы можете навигировать по различным страницам, используя ссылки в навигационной панели. Каждый рецепт можно кликнуть для просмотра детальной информации. Используйте поисковую строку для поиска рецептов по ключевым словам.

## Известные проблемы

- Неполная обработка данных API, что может вызвать проблемы с отображением компонентов.
- Внешний вид вебсайта нуждается в небольшой доработке
- Дизайн приложения нуждается в доработке для оптимального отображения на мобильных устройствах.
